# README #

KEYWORD SEARCH DEMO

## What is this repository for? ##

* Keyword search demo page

## How do I get set up? ##

* 필수설치 : Pipenv

```
$ pip install pipenv
```

* Pipenv 설치 후
```
$ pipenv sync
```
* Pipenv 가상환경 실행
```
$ pipenv shell
```
* package 설치
```
$ pipenv install <패키지명>
```
* flask 서버 구동
```
$ flask run
```

* keyword searching demo
```
$ sh ./app/static/searchingKeyword.sh "검색어"
```
